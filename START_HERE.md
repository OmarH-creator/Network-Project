## 🎯 Common Tasks

### "I want to run a test"
```bash
python scripts/test_baseline.py --duration 30
```

### "I want to start the server"
```bash
python -m src.server
```

### "I want to start a client"
```bash
python -m src.client --device-id 1001 --duration 30
```
